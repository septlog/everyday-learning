# JavaScript 引用类型

## 创建 Object 对象的两种方式

### new 关键字

```js
var person = new Object();
person.name = 'septplus';
person.age = 18;
```

### 对象字面量

```js
var person = {
    name: 'septplus',
    age: 18
};
```

## JavaScript 可以通过方括号访问对象的属性

### 这样的好处是可以通过变量访问属性

### 除非必须使用变量访问属性，否则建议使用点表示法

```js
console.log(person.name);
console.log(person[name]);

var prop = 'name';
console.log(person[prop]);
```
